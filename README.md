# Sprint Log

Sprint Log is a personal website and blog for me, working as a senior software engineer with 4+ years of experience building scalable SaaS platforms, fintech systems, and secure browser extensions.

## Features

* Written in TypeScript using React
* Includes a blog with posts about software engineering, fintech, and security
* Uses Lucide Icons for a consistent design language
* Has a simple, responsive design for easy reading on mobile and desktop devices

## Development

* Clone the repository and run `npm install` to install dependencies
* Run `npm run dev` to start the development server
* Run `npm run build` to build the production site
* Run `npm run start` to start the production server

## Deployment

* The site is deployed to Vercel using the GitHub Actions workflow
* The site is available at [https://vaibhavkumar.in](https://vaibhavkumar.in)
